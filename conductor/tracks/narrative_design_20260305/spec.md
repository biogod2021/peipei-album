# Specification: Comprehensive Narrative Design (Scrapbook Edition)

## 1. Overview
Create a sophisticated, "Interactive Scrapbook" gallery for "Time Letter" that leverages existing photo filenames and stories as the primary narrative engine. This track focuses on the transition from a simple image grid to a curated digital narrative with high emotional impact and playful interactive elements.

## 2. Functional Requirements
- **Chapter-Driven Layout:** Use `Untitled.txt` stories as "Intro Cards" for major chapters.
- **Sticky Note Annotations:** Transform individual photo filenames into visual "sticky notes" on photos.
- **Inner-Thought Reveal:** Reveal (parenthetical) text in filenames via hover or tap.
- **Mirror Comparison:** Implement a "Then vs Now" component for comparing 2025 vs 2026 milestones.
- **Emoji Particle System:** Trigger floating emoji bursts based on keywords in text (e.g., 🌶️, 😭, 📸).
- **Foodie Highlighter:** Add a unique visual marker to photos mentioning food.

## 3. Non-Functional Requirements
- **Nostalgic Aesthetic:** Use warm textures, handwritten-style fonts, and scrapbook-like layouts.
- **Performance:** Maintain 60fps scrolling on mobile despite particle effects.
- **Manual Curation:** Allow for granular, manual placement of all assets.

## 4. Acceptance Criteria
- [ ] 5+ "Mirror Moments" implemented.
- [ ] "Sticky Note" CSS component functional.
- [ ] "Inner-Thought" reveal logic functional.
- [ ] Emoji Particle system functional.

## 5. Out of Scope
- Global search.
- Social sharing.
